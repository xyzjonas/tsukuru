# TiskniLepe3d webpage

Prismic + NuxtJS

## Development

All esential commands can be found in the justfile. When you're ready to start your project, run the following commands:

```sh
npm i
just login
just dev  # start the NuxtJS dev server
just slicemachine  # start the Prismic slicemachine server

just generate  # build the site (SSG)
```
